what is web?

The World Wide Web is a space where users connected to the Internet can share information with each other. To be precise, the Web is just one popular service on the Internet. However, nowadays, the words Internet and Web are used interchangeably, so they occupy the largest part of the Internet.

The web provides multimedia information such as text, picture, sound, and video on the Internet by connecting it in a hypertext manner. Hypertext is a document that exists on the web by inserting a reference to another document. It refers to a technology that allows multiple documents to refer to each other. In this case, a reference from within a document to another document is called a hyperlink. On the web, HTML allows anyone to create their own document. .In addition, any document on the web created in this way can be searched and accessed by anyone using the HTTP protocol.

In the web, a hypertext document written using the HTML language is called aweb page. Among these web pages, a set of web pages written with related content is called a web site. Web pages are constructed by connecting them to each other via hyperlinks. When a user continues to navigate to other Web pages by following hyperlinks contained in a Web page, this is called web surfing, when the user The program you use to browse the page is called a web browser.
